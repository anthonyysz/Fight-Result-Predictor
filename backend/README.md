# Backend

This folder holds the Python side of the project.

## Layout

- `src/historical_scraper/`
  - Scraps past UFC events, starting at a certain date through present day
- `src/upcoming_scraper/`
  - Scrapes the next upcoming event only
- `src/api/`
  - 
- `data/reference/`
  - Source files the scraper reads, such as `fighter_aliases.csv` and `rankings_history.csv`
- `data/generated/`
  - CSV outputs generated by scraper runs
- `db/`
  - PostgreSQL schema and query files
- `.env`
  - Backend environment variables and database connection settings

## Install

From the repo root:

```powershell
.venv\Scripts\pip install -e .\backend
```
## Load

From backend directory:
```powershell
Invoke-WebRequest -Uri http://127.0.0.1:8000/health -UseBasicParsing
```

To clear it:
```powershell
Stop-Process -Id 29604
```

To start it:
```powershell
$env:PYTHONPATH = (Resolve-Path .\src).Path
```
